## Getting started
1. Clone the repo
    ```
    $ git clone https://github.com/dferrreira/minicart-test
    ```
2. Install the dependencies
    ```
    $ yarn
    ```
    or
    ```
    $ npm i
    ```

3. Run the webpack-dev-server
    ```
    $ npm start
    ```
    or, build for production
    ```
    $ npm run build
    ```
4. Node version indicated **14.15.1**
